# Admin Dashboard Frontend

A modern admin dashboard built with Next.js 14, TypeScript, Bootstrap CSS, and React Query.

## Features

- 📊 Dashboard with analytics and real-time updates
- 📦 Product management
- 🛒 Order management
- 👥 Customer management
- 📝 Blog management
- 📈 Analytics and reporting
- 🔐 Authentication and authorization
- 🌓 Dark mode support
- 📱 Responsive design

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Bootstrap CSS
- TanStack React Query
- Socket.IO Client
- Recharts
- Lucide Icons

## Getting Started

### Prerequisites

- Node.js 18+
- npm/yarn/pnpm

### Installation

```bash
# Clone the repository
git clone <your-repo-url>

# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Copy environment variables
cp .env.local.example .env.local

# Start development server
npm run dev
